# Frontend

React 19 SPA, собирается Vite 8, один файл `App.jsx` (~700 строк).

## Сборка

```bash
cd frontend
npm install      # зависимости
npm run dev      # dev-сервер :5173 с proxy на :5001
npm run build    # production → dist/
```

## Vite конфигурация

```js
// vite.config.js
server: {
  proxy: {
    '/api': { target: 'http://localhost:5001', changeOrigin: true }
  }
}
```

В development API-запросы проксируются на Flask. В production Nginx делает то же самое.

## Компоненты

### App (корневой)

Состояния:
- `user` — текущий пользователь (null = не авторизован)
- `text` — ввод для расчёта
- `results` — результат расчёта
- `history` — история расчётов (группировка по датам)
- `showAdmin` — показать админку
- `showHint` — показать подсказку формата

Логика:
1. При загрузке проверяет токен через `GET /api/me`
2. Если не авторизован → `LoginScreen`
3. Если `showAdmin` → `AdminPanel`
4. Иначе → основной интерфейс расчёта

### LoginScreen

Форма логин/пароль → `POST /api/login` → сохраняет токен.

### AdminPanel

3 вкладки:

**Пользователи** (admin):
- Таблица: аватар, имя, логин, роль, действия
- Загрузка/удаление аватаров
- CRUD пользователей с формой
- Выбор роли: менеджер/редактор/админ

**Теплоходы** (admin + editor):
- Поиск по названию/причалу
- Таблица: название (ссылка), причал, уборка
- Inline-редактирование: ссылка, причал, уборка, подготовка, разгрузка

**Синхронизация** (admin):
- Счётчик теплоходов, дата последней синхронизации
- Кнопка "Синхронизировать с сайтом"
- Статус-сообщение на 8 секунд

### UserAvatar

Показывает аватарку или плейсхолдер с первой буквой имени.

### CopyButton

Копирование текста в буфер обмена с фидбеком "Скопировано!" (2 сек).

### Адаптивность

Брейкпоинт: `500px`

- `≤ 500px`: single column, полноширинные кнопки, стек-лейаут
- `> 500px`: grid для форм, inline кнопки
